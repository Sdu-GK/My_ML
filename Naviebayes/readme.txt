Bernoulli_Model.py与Multinomial_Model.py均为朴素贝叶斯对文本分类的应用程序，参考于文件naviebayes.py
两个程序应用说明如下：
1.Bernoulli_Model.py使用伯努利模型，计算公式如下：
P(C) = 类C下文件总数 / 整个训练样本的文件总数
P(Word|C) = (类C下包含单词Word的文件数 + 1) / (类C下文件总数 + 2)

2.Multinomial_Model.py使用多项式模型，计算公式如下：
P(C) = 类C下单词总数 / 整个训练样本的单词总数
P(Word|C) = (类C下单词Word在各个文档中出现的次数之和 + 1) / (类C下单词总数 + 训练样本中出现的单词种类数)

3.两个程序在做分类时，将训练文件放在Tarin文件夹下，每一类放在一个文件夹下，文件夹命名为0,1,2,...同时每个类别的文件夹下的文件
命名为0.txt,1.txt,2.txt... ,程序将文件中的所有训练数据分为训练集和测试集，用训练集训练，测试集测试，并返回正确率。如果要对一
各文本进行分类，可将文件放在Test文件夹下，命名为0.txt,1.txt,2.txt...程序返回一个列表，列表中对应元素值表示对应文件的类别。
